//What is Bom vs Dom in JS? Document

Bom(Browser object model) Def:- The BOM allows JavaScript to interact with the browser itself. It provides objects that represent the browser window and its components. -> Window Obj -> Navigator Obj -> Location Obj Dom(Document object model) Def:- The DOM is a programming interface for HTML and XML documents. It represents the structure of the web page as a tree of objects, allowing JavaScript to manipulate the content, structure, and style of the document. -> Element Node -> Text Node -> Attributes
